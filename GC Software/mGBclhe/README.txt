MODULAR GAMEBOY COMPATIBILITY LAYER / HARDWARE EMULATOR - mGBclhe
VERSION 0.0.0

Contains:
- GUI that loads on startup of the GC
- Clock modifier that will control the chip's clock speed to match the framerate of the GB. 

DISCLAIMER:
- This program will not emulate all of the original GameBoy's properties exactly in it's current state. It contains a clock speed modifier
    to continuously match the speed of the original GB.
